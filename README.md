## Key Features

 - D for forward movement
 - SPACE for Jump
 - F1 to start the first level
 - F3 to start from the beggining of the current level
 - F5 ​Save the current game state
 - F6 ​Load the previous state
 - F9​ View colliders
 - F10​ God Mode

 - SPACE to jump
